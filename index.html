Hi, I'm Kurt Laurence P. Carcueva — a 3rd-year Information Technology student at Cebu Institute of Technology – University, expected to graduate in 2026.

Since stepping into college in 2023, I've been on a fast-paced journey through modern tech from the fundamentals of C and Java, to building responsive experiences with HTML, CSS, and JavaScript, to developing full applications with Spring Boot, React, and Python.

Alongside technical studies, I've trained in UI/UX design learning to think beyond the code and focus on how real users feel when they interact with a product.

My first independently-built frontend app, SkillSwap, is a peer-to-peer skill exchange platform I took from wireframe to deployment. It reflects everything I've learned so far: clean component architecture, thoughtful UX flows, and a commitment to building things that actually work.
